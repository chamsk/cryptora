Скачать и установить исходники JS https://nodejs.org/en/download/
Выгрузить файлы проекта.
Через командную строку перейти в директорию проета импортировать зависимость командой npm install pdf-parse
Выполнить скрипт командой  node pdfParserForDateMatch.js или pdfparserForOrderedMatch.js в зависимости от того, что ищем.
