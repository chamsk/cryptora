Скачать и установить исходники JS https://nodejs.org/en/download/
Выгрузить файл проекта selenium.js 
Скачать chrome driver https://chromedriver.chromium.org/downloads или другой драйвер.Положить в корень проекта.
Через командную строку перейти в директорию проета импортировать зависимость командой npm install selenium-webdriver
и выполнить скрипт командой  node selenium.js
